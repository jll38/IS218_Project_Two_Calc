OOPs

OOP (Object Oriented Programming), as the name implies, is a programming paradigm that utilizes objects and classes into programming. The main concepts of Object- Oriented Programming are:

  • Classes
  • Objects
  • Polymorphism
  • Encapsulation
  • Inheritance

Encapsulation - The idea off wrapping data and the methods that work on the data within one unit. This restricts the accessibility of variables and methods, preventing the accidental modification of data. Object's variables can only be changed by an object's method.

Abstraction - A blueprint for other classes. Creates a set of methods that must be created in a child class built from the abstract class.
