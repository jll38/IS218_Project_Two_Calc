Pylint

Pylint is a tool that check for Python code errors. It also looks out for code smells, which are characteristics in code that indicates a deeper problem in the code, and if not delt with, could lead to a refactor being necessary. Pylint can recommend suggestions about how particular blocks of code can be refactored, along with details about the complexity of the code.
